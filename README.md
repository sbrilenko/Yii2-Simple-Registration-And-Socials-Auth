Yii 2 Basic Registration and Social Auth Project
============================
- by using [https://github.com/amnah/yii2-user](https://github.com/amnah/yii2-user)

# Steps to run

- git clone https://github.com/sbrilenko/Yii2-Simple-Registration-And-Socials-Auth.git
- cd Yii2-Simple-Registration-And-Socials-Auth
- composer install
- edit file config/web.php authClientCollection:
   - part authClientCollection->clients and add your own socials keys and secrets
